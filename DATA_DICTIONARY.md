# APGI Seed Dataset — Data Dictionary

Variable codebook for all `.npz` archives in `data/seeds/`.  
Each archive is accompanied by a `.csv` summary (same stem, same directory) for readers
using R, MATLAB, or Julia.  Shapes assume default generation parameters; change with
`--dataset` flags in `data/generate_seeds.py`.

All datasets are generated by `data/generate_seeds.py` with `MASTER_SEED = 2025`.  
Checksums: `data/checksums.sha256`.  Zenodo DOI: `10.5281/zenodo.XXXXXXX`.

## sim1_ignition_dynamics.npz  (~1.5 MB)

**Used by:** Figure 1, `notebooks/protocol1_cardiac_eeg.ipynb`  
**Rows:** 10 000 simulated trials  
**Model parameters stored in archive:** α=0.3, β=0.7, γ=0.9, κ=100

| Variable | Shape | Dtype | Units | Equation symbol | Description |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `pi_e` | (10000,) | float64 | a.u. | Πᵉ | Excitatory precision per trial |
| `z_e` | (10000,) | float64 | a.u. | zᵉ | Excitatory prediction-error magnitude |
| `pi_i` | (10000,) | float64 | a.u. | Πⁱ | Raw inhibitory precision per trial |
| `z_i` | (10000,) | float64 | a.u. | zⁱ | Inhibitory prediction-error magnitude |
| `C_metabolic` | (10000,) | float64 | a.u. | C | Metabolic cost per trial |
| `V_information` | (10000,) | float64 | a.u. | V | Information value per trial |
| `pi_i_eff` | (10000,) | float64 | a.u. | Πⁱ_eff | Metabolically gated inhibitory precision: Πⁱ·exp(−C/κ) |
| `S_t` | (10000,) | float64 | a.u. | Sₜ | Global integration signal: Πᵉ\|zᵉ\| + Πⁱ_eff\|zⁱ\| |
| `theta_t` | (10000,) | float64 | a.u. | θₜ | Adaptive ignition threshold: α·C + β·V (rolling γ update) |
| `ignition` | (10000,) | bool | — | — | True when Sₜ ≥ θₜ |
| `cardiac_phase` | (10000,) | int64 | — | — | 0 = systole, 1 = diastole (alternating 2-trial cycles) |
| `S_t_cardiac` | (10000,) | float64 | a.u. | Sₜ | Sₜ recomputed with cardiac-phase-modulated Πⁱ |
| `ignition_cardiac` | (10000,) | bool | — | — | Ignition flag for cardiac-modulated Sₜ |

**Metadata keys** (stored as scalar arrays prefixed `_meta_`):  
`n_trials`, `alpha`, `beta`, `gamma`, `kappa`, `master_seed`,
`ignition_rate`, `ignition_rate_systole`, `ignition_rate_diastole`

**CSV summary** (`sim1_ignition_dynamics.csv`): one row per trial, columns
`trial`, `pi_e`, `pi_i`, `C_metabolic`, `V_information`, `pi_i_eff`,
`S_t`, `theta_t`, `ignition`, `cardiac_phase`.

## sim2_parameter_recovery.npz  (~0.1 MB)

**Used by:** Figure 2, `notebooks/quick_start.ipynb` (Appendix A.4)  
**Rows:** 1 000 independent MLE recovery runs  
**Convergence flag:** from `scipy.optimize.minimize` (Nelder-Mead) `success` attribute

| Variable | Shape | Dtype | Units | Description |
| :--- | :--- | :--- | :--- | :--- |
| `beta_true` | (1000,) | float64 | a.u. | Generating β value for each run |
| `beta_hat` | (1000,) | float64 | a.u. | MLE-recovered β̂ |
| `pi_i_true` | (1000,) | float64 | a.u. | Generating Πⁱ value for each run |
| `pi_i_hat` | (1000,) | float64 | a.u. | MLE-recovered Π̂ⁱ |
| `converged` | (1000,) | bool | — | True when scipy Nelder-Mead strict xatol/fatol criterion met |
| `converged_residual` | (1000,) | bool | — | True when NLL/n_trials < 2.0 (robust criterion for flat NLL surfaces) |

**Metadata keys:** `n_runs`, `n_trials_per_run`, `beta_range_lo/hi`, `pi_i_range_lo/hi`,
`pearson_r_beta`, `pearson_r_pi_i`, `criterion_met_beta`, `criterion_met_pi_i`,
`convergence_rate_optimizer`, `convergence_rate_residual`,
`convergence_source` (= `"scipy_nelder_mead_success_flag"`),
`convergence_residual_criterion` (= `"nll_per_trial_lt_2.0"`), `master_seed`

**Note on convergence rates:** The strict optimizer flag (`converged`) reflects
simultaneous satisfaction of `xatol=1e-6` and `fatol=1e-6`, which rarely fires on
flat NLL surfaces even when estimates are accurate (Pearson r > 0.96).
`converged_residual` is the recommended criterion for downstream analyses.

**CSV summary** (`sim2_parameter_recovery.csv`): one row per run, columns
`run`, `beta_true`, `beta_hat`, `pi_i_true`, `pi_i_hat`, `converged`, `converged_residual`.

**Used by:** Figure 3, `notebooks/protocol2_somatic_agent_sim.ipynb`

| `lnn_ignition` | (20, 500) | bool | — | True when output_norm > 95th-percentile threshold per seed |

**Metadata keys:** `n_seeds`, `n_timesteps`, `n_inputs`, `n_hidden`, `n_outputs`,
`dt_ms`, `master_seed`

**CSV summary** (`sim3_liquid_network.csv`): per-seed statistics only (20 rows) —
`seed_idx`, `spectral_radius`, `mean_output_norm`, `sd_output_norm`, `ignition_rate`.

## sim6_bifurcation.npz  (~0.05 MB)

**Used by:** Figure 6, `scripts/APGI_LNN_Bifurcation_Analysis.py`  
**Rows:** 25 simulated subjects  
**CSD variable:** ignition ratio r_t = Sₜ/θₜ

| Variable | Shape | Dtype | Units | Description |
| :--- | :--- | :--- | :--- | :--- |
| `ac1_baseline` | (25,) | float64 | — | Lag-1 autocorrelation of r_t in baseline window [0, T_BASE) |
| `ac1_pre_ignition` | (25,) | float64 | — | Lag-1 autocorrelation of r_t in pre-ignition window [T_TOTAL−80, T_TOTAL) |
| `eigenvalue_real_max_baseline` | (25,) | float64 | — | Dominant real eigenvalue of APGI threshold update in baseline (≈γ=0.9) |
| `eigenvalue_real_max_pre` | (25,) | float64 | — | Dominant eigenvalue near ignition (approaches 1 as r_t → 1) |
| `csd_ratio` | (25,) | float64 | — | SD(r_t, pre-ignition) / SD(r_t, baseline); pre-registered criterion ≥ 1.2 |

**Pre-registered falsification criteria:**

- AC1 increases (pre > baseline) in ≥ 20/25 subjects
- Mean CSD ratio ≥ 1.2

**Metadata keys:** `n_subjects`, `t_baseline`, `t_ramp`, `t_pre_ignition_len`,
`t_total`, `alpha`, `beta`, `gamma`, `kappa`, `csd_variable`, `window_design`,
`falsification_criterion_ac1_n`, `falsification_criterion_csd`,
`ac1_increases_count`, `mean_csd_ratio`, `master_seed`

**CSV summary** (`sim6_bifurcation.csv`): one row per subject (25 rows) —
`subject_idx`, `ac1_baseline`, `ac1_pre_ignition`, `ac1_increase`,
`csd_ratio`, `eigenvalue_baseline`, `eigenvalue_pre`.

d = np.load("data/seeds/sim5_doc_biomarker.npz", allow_pickle=True)
